# Data and output licensing

The MIT license in `LICENSE` applies to original source code and repository documentation.

The raw literature-derived data files were obtained from the supplementary data associated with Tian et al. (2021), *Machine Assisted Experimentation of Extrusion-Based Bioprinting Systems*, Micromachines 12(7), 780. The original source terms continue to apply; this repository does not relicense third-party data.

Processed datasets, audit records, derived tables, figure source data, and generated figures should be cited with both the present repository/archive and the original data source. Before public release, the authors should confirm that redistribution of the raw source files is consistent with the source publication's terms.
